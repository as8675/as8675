  CI/CD pipeline and auto-scaling infrastructure maintaining 100% availability during traffic spikes.

- **[Crime Atlas](https://github.com/as8675/Crime-Atlas)** — MERN geospatial platform rendering 1M+ crime records
  via Google Maps API, with optimized Node.js aggregation queries delivering spatial insights in under 2 seconds and
  interactive features like Street View imagery and real-time comments · [Live](https://crimeatlas.vercel.app/)

- **[Amazon Sentiment Dashboard](https://github.com/as8675/Amazon-sentiment-analysis)** — AWS-provisioned React

  processing 15,000+ reviews via Amazon Comprehend.

  patient records into Neo4j with optimized batch transactions (10,000 rows/batch) and high-performance Cypher queries
  streaming complex relationship paths into a D3.js visualization interface · [Live](https://neo4j-project-as8675.onrender.com/)

- **[LogiMasters IMS](https://github.com/as8675/LogiMasters)** — Real-time inventory management platform with
  optimized PostgreSQL schemas for high-frequency updates, 98% unit/integration test coverage via GitHub Actions

- **[Space Trivia App](https://github.com/as8675/Space-App-)** — Android app with 500+ Google Play downloads,
